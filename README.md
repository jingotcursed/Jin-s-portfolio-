# 🌐 Jin's Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript.  
It is a modern, responsive, and animated website designed to showcase my skills and projects.

---

## ✨ Features

- Modern UI design with gradient background  
- Smooth scrolling navigation  
- Animated snow background effect  
- Profile image with golden glow effect  
- Fully responsive for mobile and desktop  
- AOS scroll animations  
- Projects showcase section  
- Contact form (email + WhatsApp integration)  
- Sticky navigation bar
